Sapphire

# Description
ショウジョウバエ集団活動量ビューア。  
画像解析により個体ごとの活動量の定量化、ライフステージ転換点の自動検出が可能です。

- dash-renderer (0.24.0)

- keras (2.2.4)
- numpy (1.16.5)

- Pillow (6.1.0)
